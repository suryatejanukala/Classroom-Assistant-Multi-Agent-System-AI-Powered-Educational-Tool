REQUEST TYPES

  • LESSON_PLAN → LessonPlanAgent
  • WORKSHEET → WorksheetAgent
  • GRADING → GradingAgent
  • PERFORMANCE_ANALYSIS → StudentPerformanceTool
  • MEMORY_SUMMARY → MemoryService
  • METRICS_REPORT → MetricsService
REQUEST PROCESSING FLOW
1. Teacher Input
↓
2. ClassroomAssistantApplication.processRequest()
• Start timer
• Set metrics: active_sessions = 1
↓
3. OrchestratorAgent.process()
• analyzeRequest() → RequestType
• Route to appropriate handler
↓
4. Specific Agent Processing
• LessonPlanAgent → Generate lesson plan
• WorksheetAgent → Generate worksheet
• GradingAgent → Process grading
↓
5. Memory & Metrics Update
• Store result in ClassroomMemoryService
• Update MetricsService counters
↓
6. Response Generation
• Format output
• Record timing metrics
• Set active_sessions = 0
↓
7. Return to Teacher

TIMING METRICS

  • LessonPlanAgent: 2+ seconds (long-running)
  • WorksheetAgent: ~25ms (fast)
  • GradingAgent: Variable based on input size
  • Memory operations: <10ms
  • Metrics recording: <5ms
